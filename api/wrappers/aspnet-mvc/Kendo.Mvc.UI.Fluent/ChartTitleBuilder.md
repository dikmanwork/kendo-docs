# Kendo.Mvc.UI.Fluent.ChartTitleBuilder
Defines the fluent interface for configuring the ChartTitle.



## Methods

### Text(`System.String`)
Sets the title text


#### Parameters

##### text `System.String`
The text title.




#### Example (ASPX)
    <% Html.Kendo().Chart()
        .Name("Chart")
        .Title(title => title.Text("Chart"))
        .Render();
    %>


### Font(`System.String`)

#### Parameters

##### font `System.String`
