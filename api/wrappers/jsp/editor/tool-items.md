# \<kendo:editor-tool-items\>

For tools that display a list of items (fontName, fontSize, formatting), this option specifies the items in the shown list.

#### Example
    <kendo:editor-tool>
        <kendo:editor-tool-items></kendo:editor-tool-items>
    </kendo:editor-tool>
